## Prerequisites

Node.js v18.14.2
npm 9.8.1

## Installation

# 1. Clone the repository

git clone https://github.com/Martin-Johnsson/CardKeeper

# 2. Navigate to the project folder

# 3. Install dependencies

npm install

# 4. Run application

npm run dev
